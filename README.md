# 🔧 Универсальный Спрайт-Редактор v5.0

Веб-приложение для редактирования спрайт-листов и создания анимаций. Работает прямо в браузере без установки.

![Лицензия](https://img.shields.io/badge/license-MIT-blue.svg)
![Версия](https://img.shields.io/badge/version-5.0-green.svg)

## ✨ Возможности

### 🎨 Основные функции
- **Загрузка спрайт-листов** (PNG, JPEG, WebP)
- **Настройка сетки** — указание количества колонок и строк
- **Редактирование кадров** — перемещение, изменение размера, авто-кадрирование
- **Создание анимаций** — настройка последовательности кадров, FPS, режимов воспроизведения
- **Пресеты анимаций** — сохранение и управление наборами кадров
- **Экспорт/Импорт JSON** — совместимость с Unity, Godot, Phaser

### 🛠 Инструменты автоматизации
| Инструмент | Описание |
|------------|----------|
| **Auto-Crop** | Автоматическое кадрирование выбранного кадра по контенту |
| **Auto-Slice** | Автоматическое нарезание спрайтов по объектам на прозрачном фоне |
| **Auto-Detect Grid** | Авто-определение сетки на основе объектов изображения |
| **Remove Background** | Удаление фона по цвету (хромакей) с настройкой допуска |

### ⌨ Горячие клавиши
| Клавиша | Действие |
|---------|----------|
| `Ctrl+Z` | Отменить действие |
| `Ctrl+Y` / `Ctrl+Shift+Z` | Повторить действие |
| `Стрелки` | Сдвиг кадра на 1px |
| `Delete` / `Backspace` | Сброс кадра |
| `Пробел` | Пауза/воспроизведение анимации |
| `+` / `-` | Увеличить/уменьшить масштаб |
| `Ctrl+Клик` | Добавить кадр в анимацию |
| `Shift+Перетаскивание` | Создать новый кадр |

## 🚀 Быстрый старт

### Вариант 1: Открыть файл
Просто откройте `sprite_editor.html` в любом современном браузере.

### Вариант 2: Локальный сервер (рекомендуется)
```bash
# Python 3
python -m http.server 8000

# Node.js (npx)
npx serve .

# PHP
php -S localhost:8000
```
Перейдите по адресу `http://localhost:8000`

## 📖 Как использовать

### 1. Загрузка изображения
Нажмите **📁 Загрузить картинку** и выберите файл спрайт-листа.

### 2. Настройка сетки
Укажите количество **колонок** и **строк** для разделения изображения на кадры.

### 3. Редактирование кадров
- **Клик** — выбрать кадр
- **Перетаскивание за центр** — переместить кадр
- **Перетаскивание за угол** — изменить размер
- **Auto-Crop** — автоматически обрезать кадр по контенту

### 4. Создание анимации
1. В поле **Играть кадры** укажите индексы через запятую (например: `0,1,2,3`)
2. Настройте **FPS** и **Режим** (Loop, Ping-Pong, Once)
3. Используйте **Менеджер Анимаций** для сохранения пресетов

### 5. Экспорт данных
- **📄 Скачать JSON** — экспортировать данные кадров и анимаций
- **📂 Импорт JSON** — загрузить ранее сохранённые данные

## 🔧 Форматы экспорта JSON

Приложение поддерживает экспорт в форматах:
- **Generic Atlas** — универсальный формат
- **Unity** — для Unity Sprite Atlas
- **Godot** — для Godot Engine
- **Phaser** — для Phaser Framework

## 💡 Советы

- **Auto-Slice** лучше всего работает с изображениями на прозрачном фоне
- Для точной настройки используйте **стрелки клавиатуры**
- Включите **Показать сетку** для визуального контроля
- Данные автоматически сохраняются в LocalStorage браузера

## 🖥 Совместимость

- **Браузеры:** Chrome, Firefox, Edge, Safari (последние версии)
- **ОС:** Windows, macOS, Linux
- **Требования:** Поддержка ES6+, Canvas API, LocalStorage
